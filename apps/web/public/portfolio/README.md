# Portfolio images

Положите сюда реальные фото работ EKSAAL NAIL по следующей схеме именования:

```
nail-1.jpg
nail-2.jpg
nail-3.jpg
nail-4.jpg
nail-5.jpg
nail-6.jpg
```

Рекомендации к фото:
- портретная ориентация (соотношение примерно 3:4);
- не менее 1200px по длинной стороне;
- вес файла до ~500KB (для быстрой загрузки в Telegram Mini App).

Пока файла нет по нужному пути — карточка в галерее покажет аккуратную заглушку вместо битой картинки, ничего не сломается.

Чтобы добавить больше 6 работ или изменить подписи — расширьте массив в
`apps/web/src/pages/client/gallery/portfolioData.ts` под новые имена файлов.
